# /perf

Ask a k6 or load testing question.

Usage: `/perf <question>`

Delegate to the `perf-engineer-expert` subagent with the question. It has domain expertise in:
- k6 executors and their semantics
- The externally-controlled executor and live control
- xk6-kv for shared state
- http_req_duration measurement
- Load testing patterns (smoke, load, stress, spike, soak, breakpoint)
- OpenSynapse's specific architecture for live VU, RPS, and duration control

Present the answer to the user.
